# Baseline-Classical-Statistical-Models-

# Predictive Maintenance using Logistic Regression

# Project Overview
This project builds a statistical model to predict **machine failure in a manufacturing environment** using operational machine data. The goal is to identify factors that increase the probability of machine breakdown and support predictive maintenance.

# Model Used
A **Logistic Regression model** was implemented because the target variable is binary (failure vs. no failure). The model estimates the probability of machine failure based on machine operating conditions.

# Model Evaluation
The model performance was evaluated using :
 Accuracy
 Precision
 Recall
 F1 Score
 
 These metrics help measure how well the model detects machine failure cases.

# Key Insights
 High **rotational speed and torque** increase the likelihood of machine failure.
 **Tool wear** significantly contributes to machine breakdown.
 **Temperature variations** impact machine performance.
 Stable machine operation reduces failure risk.

# Business Impact
The model supports **predictive maintenance strategies** by identifying early warning signs of machine failure. This helps reduce downtime, lower maintenance costs, and improve operational efficiency.
